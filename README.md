## Segment-Anything-LangSplat

This repository contains a **lightly modified version** of [Segment-Anything-LangSplat](https://github.com/minghanqin/segment-anything-langsplat), tailored for integration into the [SceneDiff](https://github.com/yuqunw/scenediff) project.
This modification is built upon the [**Segment Anything Model (SAM)**](https://github.com/facebookresearch/segment-anything).

### **Installation & Setup**

Please follow the original repository of [SAM](https://github.com/facebookresearch/segment-anything) for initial installation and dependency setup.

**Note:** Any project-specific configurations or scripts related to **SceneDiff** will be documented within the main [SceneDiff repository](https://github.com/yuqunw/scenediff).
